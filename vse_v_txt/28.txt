ЦЕЛИ И ЗАДАЧИ НАСТОЯЩЕЙ ИНСТУКЦИИ
         Цель настоящего документа – предоставить пользователю, не
имеющему опыта работы с унифицированной системой управления ресурсами
предприятия на базе ПО «1С ERP 2.0», пособие, с помощью которого он
сможет овладеть основными приемами работы с системой в соответствии с
унифицированными бизнес-процессами.


         Задача - описать последовательность этапов унифицированного бизнес-
процесса и основные приемы работы с объектами системы.

         1.3 Описание этапов бизнес-процесса

               Казначей: Формирование списка документов для выгрузки

         Для отправки документов для оплаты в РЦК в системе предусмотрена
обработка «Клиент-банк РЦК». Обработка доступна в разделе Казначейство
 Сервис  Клиент-банк РЦК (Рисунок 1).




Рисунок 1. Местонахождение обработки «Клиент-банк РЦК»


         Обработка «Клиент-банк РЦК» предназначена для загрузки и выгрузки
данных по расчетным счетам организации. Обработка имеет три закладки:
         - Выгрузить документы – на закладке отражаются документы «Списание
безналичных денежных средств», которые в дальнейшем будут выгружены в
РЦК;
         - Загрузить документы – на закладке отражаются документы списания с
расчетного счета и поступления на расчетный счет полученные из РЦК.
         - Документы для проверки – на закладке отражены документы
поступления денежных средств.
         Для настройки обработки «Клиент-банк РЦК» необходимо указать
Организацию, Банковский счет организации и период для подбора документов
для выгрузки. При заполнении периода возможно выбрать только один
конкретный день. Также можно проставить флаг «НДС детально» для
детального отображения НДС. После установки настроек, на закладке
«Выгрузить         документы»   по     команде      «Заполнить»         табличная        часть
заполняется документами, согласно установленным отборам. На рисунке 2
представлена обработка «Клиент банк (РЦК)».
